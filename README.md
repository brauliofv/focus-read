# FocusRead - Extensión de Lectura Rápida para Chrome

FocusRead es una extensión para Google Chrome diseñada para facilitar la lectura en la web. Permite al usuario seleccionar cualquier bloque de texto y leerlo palabra por palabra a una velocidad configurable, utilizando un resaltado de alto contraste que se adapta a temas claros y oscuros.

## Características del MVP

- **Lectura Secuencial:** Resalta palabra por palabra el texto seleccionado.
- **Velocidad Ajustable:** Configura las Palabras Por Minuto (PPM) desde un popup.
- **Control Intuitivo:** Haz clic en cualquier lugar para pausar o reanudar la lectura.
- **Resaltado Adaptativo:** Usa un color amarillo para fondos claros y azul para fondos oscuros.
- **Feedback Visual:** La palabra resaltada cambia a naranja cuando la lectura está en pausa.
